## 这是一个测试页面，用于记录 opts.host 的开发日记

由于我自己想偷懒，索性就直接用了 Docsify 在你们的浏览器上渲染 markdown 了...不过就个人而言，我还是很喜欢这种文件资源管理器的风格的（主要是不用适配，无脑拖拽就行了）。不过这是我第一次用，Docsify 的特性还在探索。

- **特别适合写点什么的懒癌晚期**,在没有必要的情况下和重量级说拜拜了。
- 所以压根没有考虑兼容性。
- 也没有多国语言支持和残疾人辅助。
- 性能较差的浏览器可能没办法直接使用。

这个页面不是长期固定存在的，因为还在帮忙实验 opts.host 的某些功能。大概是长期域名的原因，开发效率基本上没有...移植什么的好麻烦的...

_Docsify_ 依旧处于试用状态，在没有完全完成网站开发的情况下，它太适合过渡了。安全管控、成本审计、系统安全嗅探什么的都不用仔细考虑。
