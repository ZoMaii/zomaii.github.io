# &#x1F41B; 这是一个测试页面，用于记录 opts.host 的开发日记

由于我自己想偷懒，索性就直接用 Docsify 在你们的浏览器上渲染 markdown 了...不过就个人而言，我还是很喜欢这种文件资源管理器的风格<ruby>的<rt>d&#x012B;<rt></ruby>（主要是不用适配，无脑拖拽就行了）。

- **特别适合写点什么的懒癌晚期**,在没有必要的情况下和重量级说拜拜了。
- 所以压根没有考虑兼容性。
- 也没有多国语言支持和残疾人辅助。
- 性能较差的浏览器可能没办法直接使用。

_Docsify_ 目前来说只是临时过渡使用的，<s>在没有完全完成网站开发的情况下</s>。但由于 opts.host 是长期域名，大家开发都没有什么欲望（去提升效率）......
